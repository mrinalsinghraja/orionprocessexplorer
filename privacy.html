3. On-device AI (Apple Intelligence)

All AI features in the App — including process explanations, the Orion Assistant chatbot, kill risk assessments, health summaries, and process name decoding — are powered by Apple Intelligence via the FoundationModels framework.

This means all AI inference runs entirely on your Apple Silicon device. No prompts, no process names, no system data, and no AI-generated responses are ever sent to any external server — including Apple's servers or the Developer's servers.

For more information on Apple Intelligence and privacy, see apple.com/privacy.

📡
4. Network access

The App requests network access for one purpose only: to display your public IP address on the Network screen. This is a standard read-only request to a public IP lookup service. No personal data is included in or associated with this request.

All other app functionality — process monitoring, system info, AI features, storage analysis — operates entirely offline with no network requests.

🔐
